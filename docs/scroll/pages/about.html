genpage create a website instantly without using a framework focusing on simplicity and flexibility. What you need to know is HTML and CSS to publish your webpage on the internet.

Icons

As you see in your html, it uses fontawesome icons in default. Thus, you can use any icon you want. You can find icons at fontawesome site.

<link rel="stylesheet" href="//cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.4/css/all.min.css" />
